# KOTARO UCHIKI — Portfolio

内木小太郎のポートフォリオサイトです。
空間・プロダクト・体験を横断し、人が自然と動き出す「小さなきっかけ」を紹介します。

## Structure

- `index.html` — コンテンツとページ構造
- `styles.css` — レイアウト、色、レスポンシブ、モーション
- `script.js` — スクロール表示、進捗、パララックス、モバイルメニュー
- `assets/images` — 作品・プロフィール写真
- `assets/docs` — PDFポートフォリオ

外部ライブラリを使わず、HTML / CSS / JavaScriptだけで構成しています。
